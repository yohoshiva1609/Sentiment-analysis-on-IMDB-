In this, Repository we do some Sentiment analysis like classifying a movie review is positive or negative using Logistic classifier.

Download code file in Repository.

NOTE :
We use the pickle file to fast processing.
We use TF-IDF ( Term frequency-inverse document frequency) model to convert our text to numerical data after we use the numerical data set for the logistic model to 
analysis the movie review is positive or negative
